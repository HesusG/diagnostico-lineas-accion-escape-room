# Prompts para Generación de Imágenes - Escape Room Diagnóstico

Este documento contiene prompts detallados para generar imágenes con IA en **estilo Pixel Art 16-bit**, inspirado en RPGs clásicos como Final Fantasy VI y Chrono Trigger.

---

## Estilo General

**Prompt base para todas las imágenes:**
> Pixel art, 16-bit style, inspired by SNES RPG games like Final Fantasy VI and Chrono Trigger. Clean pixels, limited color palette (16-32 colors), no anti-aliasing, retro gaming aesthetic.

## Herramientas Recomendadas para Generación

1. **Midjourney** - Usar `--style raw --stylize 50` para mejor control de pixel art
2. **DALL-E 3** - Especificar claramente "16-bit pixel art, no smoothing, visible pixels"
3. **Stable Diffusion** - Usar modelos especializados en pixel art como "Pixel Art XL"
4. **Aseprite** - Para retoques manuales post-generación

---

## Notas de Implementación

1. Todas las imágenes deben tener fondo transparente donde sea apropiado (NPCs, objetos)
2. Mantener consistencia de paleta entre todas las imágenes
3. Los fondos de habitación deben dejar espacio para elementos de juego (puertas, objetos)
4. Escalar imágenes con "nearest neighbor" para mantener pixels nítidos
5. Formato PNG para preservar transparencia y calidad
